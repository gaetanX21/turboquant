# TurboQuant

A simple and efficient PyTorch implementation of the TurboQuant quantization algorithm ([arXiv](https://arxiv.org/abs/2504.19874)). 

This repository provides the core logic for TurboQuant (`turboquant.py`) along with a visualization script (`plot.py`) to analyze the error statistics and distributions generated by the quantization process.

## 📦 Requirements

Before running the scripts, make sure you have the following dependencies installed:

```bash
pip install torch matplotlib seaborn fire
```

## 🚀 Usage

### Core Implementation
The main algorithm is implemented in `turboquant.py`. You can import and use the `TurboQuant` module directly in your own PyTorch projects:

```python
from turboquant import TurboQuant

# Initialize TurboQuant with dimension (d) and bits (b)
tq = TurboQuant(d=1024, b=4)

# Quantize and dequantize your tensors
# X_hat = tq.dequantize(*tq.quantize(X))
```

### Analyzing Error Statistics (`plot.py`)

The repository includes `plot.py`, a convenient CLI tool built with [Python Fire](https://github.com/google/python-fire) to visualize coordinate differences and vector distances of the quantized tensors. It automatically generates and saves statistical distribution plots.

You can run `plot.py` right from the command line. By default, it runs the simulation for 10,000 vectors (`N=10_000`) of dimension 1024 (`d=1024`) across bit widths of 1, 2, 3, and 4 (`b_list=[1, 2, 3, 4]`).

**Run with default parameters:**
```bash
python plot.py
```

**Run with custom parameters:**
Because it uses `fire`, you can easily override the default arguments `b_list`, `d`, and `N`. 

```bash
# Test for a specific dimension and number of samples
python plot.py --d=512 --N=5000

# Test for specific bit widths
python plot.py --b_list="[2, 4, 8]"
```

### 📊 Output
When you execute `plot.py`, it calculates the mean ($\mu$) and standard deviation ($\sigma$) of both coordinate errors and vector distances. It will automatically create a `figures/` directory (if it doesn't already exist) and save the generated distribution plots as PNG files formatted like `b=4_d=1024_N=10000.png`.
